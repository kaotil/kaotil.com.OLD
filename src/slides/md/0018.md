# My aws - nginxセキュリティ設定編

@kaotil

2016/00/00



```
/etc/nginx/nginx.conf
Nginxのバージョン番号をエラーページとServer headerに含まれないようする
server_tokens off;

/etc/php.ini
// HTTPヘッダにPHPのバージョンを一緒に出さないようにする
expose_php = Off
```
